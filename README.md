# Markdown Previewer
freeCodecamp front end libraries project

## Tech Stack
* HTML
* CSS
* React 
* Marked.js
* Bootstrap

## Instructions 
1. Clone this repo or download the zip file. 
2. Run ```npm start``` to start the app. 
3. Type your text in the editor and will appear in the previewer.
